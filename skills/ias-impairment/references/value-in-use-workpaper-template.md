# Value-in-Use Workpaper Template

Use this template only after an impairment indicator is identified or an annual test is required.

## Header

| Field | Required |
|---|---|
| Entity and reporting date | yes |
| Asset or CGU tested | yes |
| Framework | yes |
| Triggering indicator | yes |
| Prepared by and reviewed by | yes |
| Forecast approval reference | yes |

## Cash-flow model

| Input | Rule |
|---|---|
| Forecast period | Normally board-approved budget period, commonly up to 5 years unless support exists for longer. |
| Cash-flow basis | Future cash inflows and outflows from continuing use and ultimate disposal. |
| Financing cash flows | Exclude financing cash flows. |
| Income tax cash flows | Use pre-tax cash flows with pre-tax discount rate, or document equivalent consistency. |
| Growth rate | Must not exceed long-term market support without reviewer approval. |
| Discount rate | Asset-specific or CGU-specific rate reflecting time value and risks not already in cash flows. |

## Calculation

| Line | Amount |
|---|---:|
| Carrying amount tested | 0 |
| Fair value less costs of disposal | 0 |
| Value in use | 0 |
| Recoverable amount: higher of FVLCD and VIU | 0 |
| Impairment loss, if carrying amount exceeds recoverable amount | 0 |

## Sensitivities

- Discount rate movement.
- Revenue or volume downside.
- Gross margin downside.
- Terminal growth downside.

## Pass criteria

- Forecasts reconcile to approved budgets or a documented forecast pack.
- Assumptions are internally consistent and not double counted.
- Impairment allocation does not reduce assets below permitted floors.
- Goodwill impairment reversal is blocked.
